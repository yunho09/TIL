# IP란?

- IP는 네트워크에서 컴퓨터나 서버를 구분하기 위한 주소이다.

인터넷에서 어떤 서버에 접속하려면 그 서버의 위치를 알아야 하는데 이때 사용하는게 IP이다.

## 공인 IP
  
- 공인 IP는 인터넷에서 직접 접근할 수 있는 IP 주소이다.  
  
- 외부 사용자가 내 공유기나 서버에 접근할 때 사용하는 주소이다.

## 사설 IP

- 사설 IP는 내부 네트워크에서만 사용하는 IP 주소이다.

- 집, 학교, 회사 같은 네트워크 안에서 기기들을 구분할 때 사용한다.

예시:

```
192.168.0.10
192.168.0.11
10.0.0.5
```

사설 IP는 외부 인터넷에서 직접 접근할 수 없다.

## 차이

```
공인 IP = 외부 인터넷에서 보이는 주소
사설 IP = 내부 네트워크 안에서만 쓰는 주소
```

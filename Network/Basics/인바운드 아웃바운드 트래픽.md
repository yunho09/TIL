# 인바운드 트래픽

- 외부에서 내 서버나 컴퓨터로 들어오는 트래픽  
## 예시

```
사용자 → 내 서버 80번 포트 접속
사용자 → 내 서버 443번 포트 접속
개발자 → 내 서버 22번 SSH 접속
```

# 아웃바운드 트래픽

- 내 서버나 컴퓨터에서 외부로 나가는 트래픽
## 예시

```
내 서버 → 외부 API 요청
내 서버 → 데이터베이스 서버 접속
내 서버 → 이메일 서버로 메일 전송
```

# 방화벽 기준

## 인바운드 규칙

```
외부에서 내 서버로 들어오는 것을 허용/차단
```

## 아웃바운드 규칙

```
내 서버에서 외부로 나가는 것을 허용/차단
```

대부분의 서버 환경에서는 아웃바운드는 기본 허용, 인바운드는 필요한 포트만 허용하는 경우가 많다.

# 생각
---
처음 들은 땐 어려운 건 줄 알았는데 생각보다 간단한 거여서 쉬웠다. 나중에 방화벽 설정을 할 때 필요한 인바운드와 아웃바운드만 허용해야겠다.

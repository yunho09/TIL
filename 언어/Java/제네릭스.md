# 제네릭스

## 정의

- 클래스나 메서드에서 사용할 데이터 타입을 외부에서 지정하는 문법

## 예제

```java
ArrayList<String> list = new ArrayList<>();
```

## 종류

| 타입    | 의미              |
| ----- | --------------- |
| `<T>` | 일반 타입           |
| `<E>` | 요소 (Collection) |
| `<K>` | Key (Map)       |
| `<V>` | Value (Map)     |
## 사용법

### 클래스

```java
class Box<T> {
    T value;

    void set(T value) {
        this.value = value;
    }

    T get() {
        return value;
    }
    
}

Box<String> box = new Box<>();
box.set("Hello");
```

### 메소드

```java
public <T> void print(T value) {
    System.out.println(value);
}
```

### 인터페이스

```java
interface Storage<T> {
    void save(T data);
}
```

---
## 사용하는 이유

- 타입 안정성
	- 잘못된 타입을 입력하면 컴파일 단계에서 차단
	
- 형변환 제거

# Collection 프레임워크

## 정의

- 여러 개의 데이터를 효율적으로 저장하고 관리하기 위한 클래스와 인터페이스 집합

## 핵심

| 구분    | 특징           | 대표 구현                 |
| ----- | ------------ | --------------------- |
| List  | 순서 있음, 중복 허용 | ArrayList, LinkedList |
| Set   | 순서 없음, 중복 불가 | HashSet               |
| Map   | key-value 구조 | HashMap               |
| Stack | LIFO (후입선출)  | Stack, Deque          |
| Queue | FIFO (선입선출)  | Queue, Deque          |

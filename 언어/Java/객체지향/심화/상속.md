# 상속

## 정의

- 기존 클래스의 필드와 메소드를 물려받아 새로운 클래스를 만드는 것

## 예제

```java
class Parent {
    String name;

    void say() {
        System.out.println("hello");
    }
}

class Child extends Parent {
}

public class Main {  
	public static void main(String[] args) {  
		Child c = new Child();  
		c.say(); // 부모 메소드 사용  
	}  
}
```

- `extends` → 상속
- Parent → 부모 클래스 (슈퍼 클래스)
- Child → 자식 클래스 (서브 클래스)

## 특징

- 부모 클래스에서 정의된 필드와 메소드를 물려 받음
- 새로운 필드와 메소드 추가 가능
- 부모 클래스에서 물려받은 메소드 수정 가능

## super 메소드

- 부모 클래스로부터 상속받은 필드나 메소드 및 생성자를 자식 클래스에서 참조하여 사용하고 싶을 때 사용하는 키워드

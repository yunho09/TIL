# 상속

## 정의

- 기존 클래스의 필드와 메소드를 물려받아 새로운 클래스를 만드는 것

## 예제

```java
class Parent {
    String name;

    void say() {
        System.out.println("hello");
    }
}

class Child extends Parent {
}

public class Main {  
	public static void main(String[] args) {  
		Child c = new Child();  
		c.say(); // 부모 메소드 사용  
	}  
}
```

- `extends` → 상속
- Parent → 부모 클래스 (슈퍼 클래스)
- Child → 자식 클래스 (서브 클래스)

## 특징

- 부모 클래스에서 정의된 필드와 메소드를 물려 받음
- 새로운 필드와 메소드 추가 가능
- 부모 클래스에서 물려받은 메소드 수정 가능

## super 메소드

- 부모 클래스로부터 상속받은 필드나 메소드 및 생성자를 자식 클래스에서 참조하여 사용하고 싶을 때 사용하는 키워드

## 오버로딩, 오버라이드

### 오버로딩

- 한 클래스 내에 동일한 이름의 메소드를 여러개 정의하는 것
- 단 매개변수의 개수 및 타입이 전부 동일하면 오버로딩 X

### 오버라이딩

- 부모 클래스로부터 상속받은 메소드의 내용을 변경하는 것

```java
class Animal {
    String name;
    String color;

    public void cry() {
        System.out.println(name + " is crying.");
    }
}

class Dog extends Animal {
	//생성자
    Dog(String name) {
        this.name = name;
    }
	// 오버라이딩
    @Override
    public void cry() {
        System.out.println(name + " is barking!");
    }
    public void swim() {
        System.out.println(name + " is swimming.");
    }
}

public class Main {
    public static void main(String[] args) {
        Animal dog = new Dog("코코");
        // Animal 클래스인 dog는 자식 클래스 Dog의 swim() 사용 불가
        // 하지만 할당 시 오버라이딩한 cry()는 적용되어 is barking이 출력된다.
        dog.cry();
    }
```

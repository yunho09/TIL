# 생성자

## 정의

- 인스턴트가 생성될 때 사용되는 인스턴스 초기화 메소드

## 예시

```java
class Phone {
    String model;
    String color;
    int price;
	
    //생성자
    Phone(String model, String color, int price) {
        this.model = model;
        this.color = color;
        this.price = price;
    }
}
```

- 클래스 이름과 동일
- 객체 생성 시 자동 실행

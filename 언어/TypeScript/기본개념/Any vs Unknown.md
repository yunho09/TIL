# 정의

* **`any` (모든 것)**  **"타입 검사를 아예 하지 않겠다"** 는 의미이다.
- **`unknown` (알 수 없음)** **"어떤 타입인지 아직 모른다"** 는 의미이다.  `any`처럼 모든 값을 할당받을 수 있지만 **타입을 확인(Type Check)하기 전까지는 변수를 사용할 수 없습니다.**

# 비교표

| 비교 항목         | any              | unknown                   |
| :------------ | :--------------- | :------------------------ |
| **의미**        | 뭐든지 허용 (검사 포기)   | 알 수 없음 (검사 필수)            |
| **프로퍼티 접근**   | 가능 (`obj.a` 허용)  | **불가능** (에러 발생)           |
| **메서드 호출**    | 가능 (`func()` 허용) | **불가능** (에러 발생)           |
| **할당 가능성**    | 모든 타입 할당 가능      | 모든 타입 할당 가능               |
| **다른 변수에 할당** | 어디든 할당 가능        | `any`나 `unknown`에만 할당 가능  |
| **사용 조건**     | 그냥 사용 가능         | **타입 좁히기(Type Guard)** 필수 |
| **안정성**       | 위험 (런타임 에러)      | 안전 (컴파일 에러)               |

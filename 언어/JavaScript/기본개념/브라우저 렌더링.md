# 렌더링 과정
1. **HTML 파싱 → DOM 생성**
2. **CSS 파싱 → CSSOM 생성**
3. **DOM + CSSOM → 렌더 트리(Render Tree)**
4. **레이아웃(Layout)**: 각 요소의 크기와 위치 계산.
5. **페인트(Paint)**: 픽셀로 변환.
6. **컴포지팅(Compositing)**: 레이어를 합쳐 최종 화면 출력.

---
# DOM (Document Object Model)
- **정의**: HTML 문서를 브라우저가 이해할 수 있도록 **트리 구조로 표현한 객체 모델**.
- **구조**: 문서 전체를 **노드(Node)로 표현.

**특징**:
- JS로 DOM을 조작하면 브라우저 화면이 즉시 변경됨.

```javascript
document.getElementById("title").innerText = "Hello DOM!";
```

# CSSOM (CSS Object Model)
- **정의**: CSS 스타일 정보를 브라우저가 이해할 수 있도록 **트리 구조로 표현한 객체 모델**.
- **구조**: CSS 규칙을 파싱해 **CSSOM 트리**를 생성.

**특징**:
- JS로 CSSOM을 조작하면 스타일이 즉시 변경됨.

```javascript
document.getElementById("title").style.color = "blue";
```

# 클로저
## 정의

- 함수가 외부 변수(스코프)를 기억해서 함수 실행이 끝난 후에도 그 변수에 접근할 수 있는 것

## 예제

```js
function outer() {
  let count = 0;

  return function () {
    count++;
    console.log(count);
  };
}

const fn = outer();

fn(); // 1
fn(); // 2
```

- 반환된 함수가 외부 변수인 count에 참조
- 참조가 남아있어서 메모리에서 삭제되지 않음

## 클로저 성립 조건

1. 내부 함수가 외부 변수를 참조해야함
2. 내부 함수가 외부에서 계속 사용되어야 함

## 사용 이유

- 상태 유지
- 데이터 보호

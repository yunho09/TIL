## 프론트엔드란?

- 프론트엔드는 사용자가 직접 보는 화면  

- 예를 들어 버튼, 입력창, 게시글 목록, 로그인 화면 같은 것들이 프론트엔드에 해당한다.

## 백엔드란?

- 백엔드는 사용자의 요청을 처리하고 데이터를 관리하는 부분  
  
- 예를 들어 로그인 처리, 게시글 저장, 데이터 조회, 회원 정보 관리 같은 일을 한다.

## 서버란?

- 서버는 프론트엔드와 백엔드가 실행되는 컴퓨터 또는 프로그램  
  
- 사용자가 브라우저로 접속하면 서버가 요청을 받고 응답을 보내준다.

## 관계  
  
- 사용자는 프론트엔드 화면을 통해 버튼을 누르거나 데이터를 입력한다.  
  
- 프론트엔드는 필요한 데이터를 백엔드에 요청하고 백엔드는 데이터를 처리한 뒤 다시 프론트엔드에 응답한다.
  
- 서버는 이 프론트엔드와 백엔드가 실제로 실행되는 환경이다.

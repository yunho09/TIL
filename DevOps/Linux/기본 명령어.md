## ls

- 현재 폴더 안에 있는 파일과 폴더 목록을 확인하는 명령어

## cd

- 다른 폴더로 이동할 때 사용하는 명령어

## mkdir

- 새로운 폴더를 만들 때 사용하는 명령어

## touch

- 새로운 파일을 만들 때 사용하는 명령어

## rm

- 파일이나 폴더를 삭제할 때 사용하는 명령어

## pwd

- 현재 내가 위치한 폴더 경로를 확인하는 명령어

## mv

- 파일을 이동하거나 이름을 바꿀 때 사용하는 명령어

## sudo

- 관리자 권한으로 명령어를 실행할 때 사용하는 명령어

## vim

- 터미널에서 파일을 수정할 때 사용하는 편집기

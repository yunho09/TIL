# 컨텍스트 엔지니어링

## 정의

- 대규모 언어 모델(LLM)이 더 정확하고 원하는 답을 하도록 설계하고 관리하는 기술

## 핵심 구성 요소

- ### 역할(Role)
	
	AI의 역할을 정의
	
- ### 목표(Task)
	
	무엇을 해야 하는지 명확히
	
- ### 조건(Constraints)
	
	출력 방식 제한
	
- ### 배경 정보(Context Data)
	
	추가 정보 제공
	
- ### 예시(Examples)
	
	원하는 출력 형태를 보여줌

## 프롬프트 엔지니어링과 비교

| **비교 항목**  | 프롬프트                         | 컨텍스트                            |
| ---------- | ---------------------------- | ------------------------------- |
| **핵심 목적**  | AI에게 '어떻게' 물어볼지 고민           | AI에게 '무엇을' 참고하게 할지 설계           |
| **주요 작업**  | 지시어 수정, 예시 제공, 페르소나부여        | 외부 데이터 검색, 중요 정보 필터링, 데이터 조립    |
| **작업 환경**  | 챗봇 대화창                       | 백엔드 시스템 및 데이터베이스                |
| **필요 역량**  | 언어적 표현력, 논리적 사고, 문맥 파악       | 데이터 전처리 기술, 시스템 아키텍처 설계         |
| **거짓말 방지** | "모르는 건 지어내지 마"라고 **규칙**을 지시함 | 아예 정답이 적힌 **근거 문서**를 시스템적으로 제공함 |
| **한 줄 비유** | 요리사에게 "맛있게 요리해 줘"라고 **주문하기** | 요리사에게 "최고급 식재료와 레시피"를 **세팅하기**  |

# 컨텍스트 엔지니어링

## 정의

- 대규모 언어 모델(LLM)이 더 정확하고 원하는 답을 하도록 설계하고 관리하는 기술

## 핵심 구성 요소

- ### 역할(Role)
	
	AI의 역할을 정의
	
- ### 목표(Task)
	
	무엇을 해야 하는지 명확히
	
- ### 조건(Constraints)
	
	출력 방식 제한
	
- ### 배경 정보(Context Data)
	
	추가 정보 제공
	
- ### 예시(Examples)
	
	원하는 출력 형태를 보여줌

## 프롬프트 엔지니어링과 비교

| 구분         | 설명                 |
| ---------- | ------------------ |
| 프롬프트 엔지니어링 | 질문 잘 만들기           |
| 컨텍스트 엔지니어링 | 질문 + 정보 + 구조 전체 설계 |

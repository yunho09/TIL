# SEO(Search Engine Optimization)
## 정의
- 웹사이트가 검색엔진 결과 페이지에서 더 높은 순위에 노출되도록 구조와 콘텐츠를 개선하는 전략입니다.
## 검색엔진 동작 원리
1 . 크롤링(Crawling)
- **웹 크롤러(Spider, Bot)** 가 인터넷을 돌아다니며 새로운 웹페이지를 발견하고 수집.
- 링크를 따라가며 페이지를 탐색하고, HTML 콘텐츠를 가져옴.

2 . 인덱싱(Indexing)
- 수집된 페이지를 **검색엔진 데이터베이스에 저장**.
- 텍스트, 이미지, 메타 태그, 키워드 등을 분석해 구조화된 형태로 기록.
- 검색엔진은 이 과정을 통해 “어떤 페이지가 어떤 주제와 관련 있는지”를 이해.

3 . 검색 및 랭킹(Search & Ranking)
- 사용자가 검색어를 입력하면, 검색엔진은 인덱스에서 관련 문서를 찾아냄.
- **랭킹 알고리즘**을 통해 결과를 순위화:
	- 키워드와 콘텐츠의 관련성
	- 페이지 권위(백링크 수, 도메인 신뢰도)
	- 사용자 경험(속도, 모바일 친화성)
	- 최신성(업데이트 시점)
- 최종적으로 **검색 결과 페이지** 에 순위별로 노출.

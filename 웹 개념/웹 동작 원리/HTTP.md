# HTTP(HyperText Transfer Protocol)
## 정의

- 웹에서 클라이언트(브라우저)와 서버가 데이터를 주고받기 위한 통신 규약
- HTML 문서뿐 아니라 이미지, JSON, 동영상 등 다양한 리소스를 전송 가능

## 동작 방식

1. **클라이언트 요청(Request)**
    
	- 브라우저가 URL을 통해 서버에 리소스를 요청
    - 요청 메시지 구성: 메서드(GET, POST 등), 헤더, 바디
    
2. **서버 응답(Response)**
    
    - 서버가 요청을 처리하고 결과를 응답
    - 응답 메시지 구성: 상태 코드(200, 404 등), 헤더, 바디
    
3. **이벤트 흐름**
    
    - 콜 스택이 비면 요청이 네트워크를 통해 서버로 전달
    - 서버는 응답을 반환하고 브라우저가 이를 렌더링
